# Бот заявок с админкой

Кейс для портфолио: Telegram-бот, через который клиент оставляет заявку, а администратор видит ее в панели и меняет статус.

## Живой проект

- Открыть проект: https://emoread23.github.io/profect745/
- GitHub Pages после включения в настройках: https://emoread23.github.io/profect745/

## Что реализовано

- форма заявки;
- список заявок;
- статусы: новая, в работе, закрыто;
- базовая аналитика;
- понятный сценарий для малого бизнеса.

## Как показывать клиентам

> Вот пример бота заявок: клиент оставляет обращение, админ получает его в Telegram и ведет статус. Такой формат подходит для услуг, ремонта, клиник, доставок и студий.

## Реальная версия

- Telegram Bot API;
- база заявок;
- роли админов;
- уведомления;
- экспорт в таблицу;
- CRM-статусы.
