# Бот заявок с админкой

Демо-проект для портфолио: Telegram-бот, через который клиент оставляет заявку, а администратор видит ее в панели и меняет статус.

## Живое демо

- Открыть демо: https://htmlpreview.github.io/?https://github.com/emoread23/profect745/blob/main/index.html
- GitHub Pages после включения в настройках: https://emoread23.github.io/profect745/

## Что показывает демо

- форма заявки;
- список заявок;
- статусы: новая, в работе, закрыто;
- базовая аналитика;
- понятный сценарий для малого бизнеса.

## Как показывать клиентам

> Вот пример бота заявок: клиент оставляет обращение, админ получает его в Telegram и ведет статус. Такой формат подходит для услуг, ремонта, клиник, доставок и студий.

## Реальная версия

- Telegram Bot API;
- база заявок;
- роли админов;
- уведомления;
- экспорт в таблицу;
- CRM-статусы.
